TxLINE feed

Real-time scores & odds stream in over SSE — the same data operators pay for, now open to builders.

NestJS core

Every event fans out as a domain event — emitted in two states so UI and settlement stay in sync.

@nestjs/event-emitter
→

Clients + Solana

Pushed to every client over WebSocket for instant UI; bets and duels settle themselves on-chain.

WebSocket + Anchor · Solana
*.during
Optimistic · live
Drives animation & instant UI the moment the pitch moves.
*.after
Authoritative
TxLINE confirms the event — settles bets and recalculates on-chain.

The same pipeline replays real historical matches — so the platform stays live even between Cup games.
